What is Turtle Tribe


With high Utility in the Turtle Tribe, aims to allow people to earn passive income while at the same time help the environment. The Turtle Tribe NFT collection aims to have the NFT’s with the most utility and value. To do so we as developers of this project came up with 3 different utilities that can help give value to TT NFT’s. Features such as Staking, a coin and a PvP game will be implemented so that TT NFT’s are not a random JPEg image!
